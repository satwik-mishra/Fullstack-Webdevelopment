NOTES

  1. Inline elements
  2. Block elements

INLINE ELEMENTS

  • New Line: They do not start on a new line;
  • they sit on the same line as adjacent content, as long as there is space.
  • Width: They only take up as much width as their content requires.

BLOCK ELEMENTS

  • They always start on a new line.
  • They take up the full widthfull width available within their parent container by default.
  • They stack vertically down the page

TO MAKE BLOCK ELEMENTS TO INLINE

  • WRITE display:block
  • display: inline-block is best hybrid model
